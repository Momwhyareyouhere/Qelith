    Flux Syntax
    
                
    
                    Entry function uses root. Helper functions use
                    glyph.
                
    
                
    glyph greet [
    emit "hello from flux\n".
]

root main [
    call greet.
    done 0.
]
    
                
    Available statements:
    
                
    emit "text\n".          # print text
line.                    # print newline
call function_name.      # call another function
loop N call function.    # repeat function call N times
done N.                  # exit with code N
    
                
    Rules:
    
                
    - Exactly one root function per file.
- Function blocks start with [ and end with ].
- Every statement ends with a dot (.).
- Comments start with #.
